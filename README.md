# Portfolio Dashboard

A modern and responsive personal portfolio dashboard built to showcase my projects, skills, and web development journey. This dashboard serves as both a professional portfolio and a live demonstration of my front-end development abilities.

## ✨ Features

* Responsive design for desktop and mobile
* Modern user interface
* Smooth navigation and layout
* Project showcase section
* About me section
* Contact section
* Clean and organized code structure
* Fast deployment with Netlify
* GitHub integration for version control

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript
* Git & GitHub

If you'd like to run the project locally:

```bash

# Open the project folder
cd dashboard
```

Then simply open `index.html` in your browser.
